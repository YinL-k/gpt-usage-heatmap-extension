# Privacy Policy

This extension tracks ChatGPT usage frequency for visualization purposes.

## Data Collection
- The extension records message submission events (usage counts and timestamps).
- No message content is collected or stored.

## Data Storage
- All data is stored locally using chrome.storage.local.
- No data is transmitted to any server.

## Data Sharing
- The extension does not sell, transfer, or share user data with third parties.
